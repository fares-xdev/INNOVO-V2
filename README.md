# INNOVO Office Furniture - Frontend

This is the frontend application for INNOVO Office Furniture, a premium office furniture provider in Egypt.

## Tech Stack

- **React** (Vite)
- **Tailwind CSS**
- **Lucide React** (Icons)
- **Shadcn UI** (Components)
- **React Router Dom** (Navigation)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
